## Installation

Install `vcf-hierarchical-network`:

```sh
npm i @vaadin/vcf-hierarchical-network --save
```

Once installed, import it in your application:

```js
import '@vaadin/vcf-hierarchical-network/vcf-hierarchical-network.js';
```
